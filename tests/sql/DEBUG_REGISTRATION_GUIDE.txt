WHAT TO DO:
============================================================

1. Open browser at http://localhost:3000/auth/register

2. Open Developer Console (F12 → Console tab)

3. Fill registration form:
   - Select user type (Service/Business/Individual)
   - Enter email
   - Enter phone (optional, format: +27821234567)
   - Enter password (8+ chars, 1 uppercase, 1 number)
   - Confirm password

4. Click "Create Account"

5. Watch console output for detailed logs
